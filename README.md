# OpenAI Chat Playground

A Streamlit-based OpenAI chatbot that allows users to dynamically configure
their API key, model, temperature, and max tokens to generate responses in real time.

This project is built to understand how LLM parameters affect output behavior
and to experiment with different OpenAI models interactively.

---

## Features

- Enter your own OpenAI API key securely
- Select OpenAI model (GPT-3.5, GPT-4, etc.)
- Control temperature for creativity
- Set max tokens for response length
- Real-time responses using Streamlit UI
- Simple, clean, and beginner-friendly architecture

---

## Tech Stack

- Python
- Streamlit
- OpenAI API
- LangChain
